---
name: PR 评论自动回复
description: 检查 PR 未回复评论时直接回复，不询问确认，但留下记录
type: feedback
---

检查 PR 评论时，若发现未回复的 root 评论，直接分析并回复，**不得向用户二次确认**。

回复后需在对话中简短列出回复记录供用户查阅（格式：`ID=xxx | 回复内容摘要`）。

**Why:** 用户认为每次回复 Copilot 评论都需要确认是多余步骤，希望自动处理后只留记录。

**How to apply:** PR 监控流程中，发现未回复评论 → 直接调用 GitHub API 回复 → 汇报「已自动回复 N 条：[列表]」。
